# Sample Document Uploads

Use these Word-compatible `.doc` files to test the document upload flow. They are intentionally realistic startup briefs and deck summaries so the extraction and enrichment prompts have useful context.

Current samples:

- `agripulse-brief.doc`
- `quantumsort-deck.doc`
- `healththread-pilot-brief.doc`
- `buildright-customer-case-study.doc`

To create another sample, duplicate one of the `.doc` files, keep the same HTML structure, and update the startup name, team, traction, and open questions. The app upload flow accepts `.doc`, `.docx`, `.pdf`, `.ppt`, `.pptx`, and images.
